## 快捷键

- 折叠展开代码

    ```shell
        ## mac下，command + k + 0，折叠所有代码块
        ## 0是代码折叠级别，同理可以换用1，2，3 等
        command + k + 0
        command + k + 1
        command + k + j ## 展开所有代码块

        ## 折叠 光标所在代码块
        command + option + [ 
        ## 展开 光标所在代码块
        command + option + ] 
    ```
